## 代表差分

### filename charset

```diff
- // NFS path strings were decoded with UTF-8 only.
+ filenameCharset = Charset.forName( getString( properties, "filename.charset", "UTF-8")) ;
+ return arguments.readString( filenameCharset) ;
+ response.writeString( name, filenameCharset) ;
```

### SETATTR

```diff
- // mode-only SETATTR was effectively ignored.
+ if( attributes.hasMode()) {
+ 	applyMode( path, attributes.getMode()) ;
+ }
```

### READDIR

```diff
- // Directory cookies were sequential per response.
+ entries.add( new DirectoryEntry( ".", directory, 1)) ;
+ entries.add( new DirectoryEntry( "..", resolveLookupPath( directory, ".."), 2)) ;
+ int cookie = READDIR_COOKIE_BASE + (name.hashCode() & 0x3fffffff) ;
```

### LINK / SYMLINK

```diff
- response.writeInt( NfsStatus.PERM) ;
+ Files.createLink( target.getPath(), sourcePath) ;
+ Files.createSymbolicLink( target.getPath(), Paths.get( linkTarget)) ;
```

### 配布バージョン

```diff
- #define AppVersion "1.0.1"
+ #define AppVersion "1.1.0"
```

```diff
-$appVersion = "1.0.1"
+$appVersion = "1.1.0"
```
