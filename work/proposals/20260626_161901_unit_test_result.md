# 単体テスト追加と実行結果

## 変更概要
- 外部ライブラリなしの単体テスト `test/jp/co/enterogawa/nfs/AllTests.java` を追加。
- テスト実行用スクリプト `scripts/test.ps1` を追加。
- Eclipse で `test` フォルダをソースフォルダとして扱うため `.classpath` を更新。
- README にコンパイル、テスト実行方法を追記。

## 実施したテスト
- XDR encode/decode の往復確認。
- RPC CALL の解析確認。
- Portmap v2 `GETPORT` の応答確認。
- MOUNT v1 `MNT` の成功応答と file handle 確認。
- NFSv2 `GETATTR` / `LOOKUP` / `READ` / `READDIR` / `STATFS` の read-only 基本動作確認。
- NFSv2 `WRITE` が `NFSERR_ROFS` を返すことを確認。
- 不正 `LOOKUP` が `NFSERR_ACCES` を返すことを確認。
- 通常ファイルへの `READLINK` が `NFSERR_INVAL` を返すことを確認。
